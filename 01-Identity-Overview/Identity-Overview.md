# Identity Overview

## Introduction

Microsoft Entra ID (formerly Azure Active Directory) is Microsoft's cloud-based Identity and Access Management (IAM) solution.

It enables organisations to manage user identities, authentication, access to applications, and security policies across cloud and hybrid environments.

Microsoft Entra ID is a core component of Microsoft 365 and plays a critical role in securing access to cloud resources.

---

## What Is Identity Management?

Identity Management is the process of creating, maintaining, and securing digital identities within an organisation.

Examples of identity management tasks include:

- Creating user accounts
- Managing user access
- Resetting passwords
- Assigning permissions
- Enforcing security controls
- Monitoring sign-in activity

These tasks are commonly performed by IT Support Analysts, Service Desk Engineers, and System Administrators.

---

## Authentication vs Authorisation

Identity management relies on two fundamental concepts:

### Authentication

Authentication is the process of verifying a user's identity.

Examples include:

- Username and password
- Multi-Factor Authentication (MFA)
- Microsoft Authenticator
- Security keys

Authentication answers the question:

> Who are you?

### Authorisation

Authorisation determines what an authenticated user is allowed to access.

Examples include:

- Group memberships
- Application permissions
- Administrative roles
- Resource access policies

Authorisation answers the question:

> What are you allowed to do?

## Microsoft Entra ID Features

Microsoft Entra ID provides a range of identity and security capabilities including:

- User management
- Group management
- Administrative role assignment
- Multi-Factor Authentication (MFA)
- Single Sign-On (SSO)
- Password management
- Sign-in monitoring
- Audit logging
- Conditional Access

These features help organisations secure access to cloud resources while maintaining a positive user experience.

---

## Microsoft Entra ID and Microsoft 365

Microsoft Entra ID acts as the identity provider for Microsoft 365 services.

When a user signs in to services such as:

- Outlook
- Teams
- SharePoint
- OneDrive

Authentication is typically performed through Microsoft Entra ID.

This allows organisations to centrally manage user identities and access permissions.

---

## Microsoft Entra ID vs Active Directory

Although Microsoft Entra ID and Active Directory share similar goals, they are different technologies.

| Active Directory | Microsoft Entra ID |
|------------------|-------------------|
| On-premises | Cloud-based |
| Windows Server role | Microsoft cloud service |
| Kerberos and LDAP | Modern authentication protocols |
| Traditional network management | Cloud identity management |
| Primarily internal resources | Cloud applications and services |

Many organisations use both technologies in hybrid environments.

---

## Why Microsoft Entra ID Matters

Modern organisations increasingly rely on cloud services and remote working.

Microsoft Entra ID helps organisations:

- Secure user identities
- Enable remote access
- Protect against unauthorised access
- Simplify administration
- Improve user productivity

As a result, Entra ID skills are highly valuable for IT support and cloud administration roles.

## Key Takeaways

- Microsoft Entra ID is Microsoft's cloud Identity and Access Management platform.
- Authentication verifies identity, while authorisation controls access.
- Entra ID integrates closely with Microsoft 365 services.
- Entra ID provides user management, security, and monitoring capabilities.
- Cloud identity management is a core skill for modern IT support professionals.
